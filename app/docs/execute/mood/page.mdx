# Mood

Productivity shouldn't be a grind, it should feel like an unstoppable joyous flow state.

Starfocus provides two moods which change the aesthetics of the app to suit your current mode of productivity.

- Chill
- Hype

Each mode also comes with a built-in playlist of matching music you can play if you so choose.

Later we intend to add Spotify integration so you can assign your own playlists to each mood.
